  • CHaserOnlineAutoとの出会い
    制作 : 埼玉県立三郷工業技術高等学校 情報技術科3年 中山 裕哉
  • CHaserOnlineAutoは、特定の型に落とし込んだCHaserOnlineアプリケーションフレームワークです。 CHaserOnlineAutoを使うと、通常のCHaserOnlineより自由に、そして簡単に作戦を練ることが容易になります。 これにより、検索の短縮、作業の効率化に集中することができます。

    CHaserOnlineAutoは、すばらしい開発者エクスペリエンスの提供に努めています。 同時に完全な依存注入、ベース操作からコマンド操作、ライブラリ拡張、サンプル実行モード、完全自動実行モードなど、 強力な機能もCHaserOnlineAutoは提供しています。

    C言語学者、アルゴリズムを理解しようとする人へ、CHaserOnlineAutoはあなたたちの学習を応援しています。 あなたがどのようなプログラムにしてくれるのか、楽しみにしています。

    下準備
    CHaserOnlineAutoを、利用するには以下のシステム要件に合格している必要があります。

    最小要件
    • OS：Linux Family(Debian系)(RedHat系) 64ビット
    • プロセッサー：Intel Core i3-4130以上
    • メモリ：2GB以上
    • グラフィックス：Intel HD Graphics 4600以上
    • ネットワーク：ブロードバンドインターネット接続
    推奨要件
    • OS：Ubuntu-Latest 64ビット
    • プロセッサー：Intel Core i3-8100以上
    • メモリ：8GB以上
    • グラフィックス：Intel UHD Graphics 630以上

    CHaserOnlineアカウントを登録しよう!
    CHaserOnlineAutoでは、最大5つまでのアカウントを登録することができます。
    アカウントの登録をするには、まず、Config/userconfig.txtを開いて自身のIDとパスワードを入力します。
